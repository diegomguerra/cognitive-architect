VYR 8.0(103) — Sincronização BLE chega ao servidor

Antes: ao parear o anel o app mostrava "Anel BLE ativo" mas as amostras coletadas via Bluetooth não chegavam ao servidor (ficavam só em memória local). VYR State funcionava com Apple Health, mas dados brutos do anel sumiam.

Agora:
- Toda sincronização BLE empurra as amostras para o servidor (ingest-biomarker-batch v34) imediatamente.
- Sincronização disparada do admin volta a funcionar (deviceId persistido em localStorage além do UserDefaults nativo).
- Caminho híbrido completo: anel → BLE → Supabase + Apple Health.

Continue usando "Conectar agora" em Integrações para parear, e "Sincronizar tudo" para forçar uma coleta.
